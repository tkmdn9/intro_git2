Gitの練習をしていきます。
基本的なワークフローを学んでいきますよ。

.git add
.git commit
.git rm
.git status
.git diff
.git rm ファイル名（gitの管理とファイルごと削除）
.git reset HEAD ファイル名（gitの管理とファイルごと削除したファイルを元に戻す）
.git checkout ファイル名
.git rm --cached ファイル名（管理からだけ外す）

【gitに登録したediterの確認方法】
git config core.editor
【gitの設定確認】
git config --list
【gitの設定ファイルを確認homeディレクトリ配下】
cat ~/.gitconfig
